Day 1 · 5月31日（周六）香港 → 关西 → 白滨 ✈️ 出发日
07:05 香港国际机场出发 → 大阪关西 (KIX)
✈️ 航班飞行约 3h50m，11:55 抵达关西机场
13:32 关西机场 → 日根野 → 白滨
🚆 JR特急「くろしお（黑潮号）」，关西空港线转纪势本线
🚆 路线：关西机场站 → 日根野站换乘 → JR白滨站
💺 建议提前指定席，くろしお全程约 2 小时；白滨是南纪熊野的门户站
15:47 JR白滨站 → 酒店
🚌 明光巴士白滨町循环巴士，前往白良滨沿岸酒店区
16:00 逛关西最大海鲜市场 · とれとれ市場
TORETORE ICHIBA，现买现吃金枪鱼、海鲜烧烤，南纪必逛的巨型鱼市场
17:30 海景 omakase + 白良滨黄昏
餐厅：SHIRAHAMA KEY TERRACE HOTEL SEAMORE；晚餐后可散步白良滨白沙海滩看落日
住宿 SHIRAHAMA KEY TERRACE HOTEL SEAMORE
🏨 白滨海崖上的温泉度假酒店，可泡断崖海景露天风吕（三段浴槽）
📍 位置：白良滨南侧海岸，靠近千叠敷、三段壁等海蚀景观
Day 2 · 6月1日（周日）白滨 → 中边路徒步 → 汤峰温泉 🥾 核心徒步日
今日是熊野古道正片：中边路「发心门王子 → 熊野本宫大社」经典段（约 7km / 4h）。这是世界遗产「纪伊山地的灵场与参诣道」中人气最高、最易走的一段，沿途经水吞王子、伏拝王子，最后从大斋原鸟居下抵达本宫大社。
08:06 白滨 → 熊野本宫大社
🚌 龙神巴士明光巴士/龙神巴士，白滨方向前往本宫大社前
09:06 熊野本宫大社前 → 发心门王子（81路）
🚌 熊野交通 81路坐到徒步起点「发心门王子」下车
🥾 81 路是从本宫大社前往发心门王子的接驳线；先坐车到高处的发心门王子，再一路下坡徒步走回本宫大社，是最省力的走法。
10:30 徒步：发心门王子 → 熊野本宫大社（7km / 4h）
中边路经典段，沿途村落、茶田与山林参道；抵达前可俯瞰大斋原（旧社地）大鸟居
📍 途经：发心门王子 → 水吞王子 → 伏拝王子 → 三軒茶屋迹 → 本宫大社
🥾 全程缓下坡为主，路况良好；备水、行动粮，山中补给点有限
⛩️ 终点：熊野本宫大社（熊野三山之一，八咫乌信仰中心），旁边大斋原立有日本最大鸟居
16:27 本宫大社 → 川汤 / 汤峰温泉
🚌 熊野交通前往汤峰温泉（湯の峰温泉）
17:00 泡温泉 + 晚餐 · 湯の峰温泉
餐饮：湯の峰温泉 売店・食堂
♨️ 汤峰温泉是日本最古老的温泉之一（约1800年），也是熊野参诣者的「汤垢离」净身地。
🈴 世界遗产名录中唯一可入浴的温泉——壶汤（つぼ湯），一次限一组、约30分钟轮流制，泉色一日数变，值得排队体验。
🥚 可在温泉源头用温泉水煮「温泉蛋」。
住宿 J-Hoppers 熊野汤之峰民宿
🏨 汤峰温泉区内的青旅/民宿，步行即达壶汤与温泉街，适合徒步后休整。
Day 3 · 6月2日（周一）新宫三社 → 那智大社 → 纪伊胜浦 ⛩️ 熊野三山日
今日集齐熊野三山另外两山——速玉大社（新宫）与那智大社，并走两段短徒步。上午在新宫市区+海滨徒步，下午走大门坂石阶参道登那智大社、看那智瀑布。
08:30 川汤 / 汤峰 → 神仓大社（新宫）
🚌 熊野交通前往新宫市，徒步起点神仓神社
09:30 城市 + 海滨徒步（约 4h）
神仓神社 → 速玉大社 → 阿须贺神社 → 王子神社 → 王子滨 → 高野坂 → 三轮崎
⛩️ 神仓神社：538 级陡峭石阶登上巨岩「ゴトビキ岩」，熊野信仰的元宫，视野极佳
⛩️ 熊野速玉大社：熊野三山之一，朱红色社殿华丽
🌊 高野坂：一段沿海的古道，可俯瞰太平洋海岸线，走到三轮崎
14:00 三轮崎 → JR那智站 → 大门坂
🚆 JR + 🚌 熊野交通前往大门坂徒步口
14:30 徒步：大门坂 → 那智大社（约 1.5h）
熊野古道最上镜的石阶参道，两侧参天杉木与苔石
🌲 大门坂：约600m、267级苔藓石阶，夫妇杉夹道，是熊野古道最经典的一段风景
⛩️ 熊野那智大社 + 青岸渡寺：三山之一，可与那智瀑布（日本落差最大瀑布，133m）+ 三重塔同框
16:00 那智大社 → 纪伊胜浦
🚌 熊野交通前往纪伊胜浦温泉港町
17:00 入住浦岛酒店 · 泡洞窟海景温泉
酒店：浦岛酒店-日升馆（ホテル浦島）
♨️ 需从胜浦港乘酒店专属船进入（本身就是体验）；招牌「忘归洞」是天然海蚀洞窟温泉，面朝太平洋、浪花拍岸，极具氛围感。
🌅 另有「玄武洞」洞窟风吕，可看日出。
19:00 晚餐：吃和牛 · 焼肉ダイニング和
纪伊胜浦市区烧肉店，徒步两天后补充能量
Day 4 · 6月3日（周二）纪伊胜浦 → 关西 → 香港 🛫 返程日
上午 纪伊胜浦半日自由活动（~07:00 起）
金枪鱼市场早市 · 弁天岛海上鸟居 · 金枪鱼料理午餐
🐟 胜浦金枪鱼市场：日本近海金枪鱼卸货量数一数二，清晨可看竞拍
⛩️ 弁天岛：胜浦港内小岛，海上鸟居，可环港散步
🍴 午餐：お食事処 大和，招牌金枪鱼生鱼片/盖饭
13:46 纪伊胜浦 → 日根野 → 关西机场
🚆 JR特急「くろしお」返回，日根野换乘关空快速/HARUKA 至 KIX
20:40 关西机场 → 香港
✈️ 航班23:55 抵达香港，行程结束
🔖 实用信息与官方链接
🚌交通要点
• 南纪熊野的交通核心是巴士：本宫、汤峰、新宫、那智、胜浦之间主要靠熊野交通与明光/龙神巴士，班次少、务必先查当日时刻表并预留换乘时间。
• くろしお（黑潮号）连接关西机场/大阪与白滨、纪伊胜浦，是进出南纪的主干特急，建议指定席。
• 若多次坐巴士，可留意「熊野交通巴士一日/二日券」等企划票是否划算。

⚠️行前提醒
  • 巴士班次稀少：南纪山区巴士一天可能只有几班，错过一班可能等 1-2 小时，务必以当日官方时刻表为准并留足换乘缓冲。
  • 徒步装备：中边路虽平易，仍建议登山鞋、雨具、行动粮与足量饮水；6 月为梅雨季，注意防雨防滑。
  • 壶汤（つぼ湯）：轮流制、一次约30分钟、限一组，旺季需排队，建议尽早前往。
  • 浦岛酒店：需乘专属接驳船进入，请留意末班船时间与办理入住时间。
  • 行李托运：徒步段建议通过 Kumano Travel 或宅急便把大件行李从白滨/本宫直送到胜浦/新宫，轻装徒步。
