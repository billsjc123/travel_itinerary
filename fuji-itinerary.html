Day 2 · 7月25日（周六）河口湖 → 御殿场 → 須走口五合目 → 上江戸屋（本八合目） ⛰️ 登山日
🧳上午：Fuji View Hotel → 御殿场（轻装优先）
✅ 主用行李方案：御殿場駅旁人工寄存（已找到公开服务信息）

位置：JR 御殿場駅步行约 1 分钟的 Mt. Fuji Gotemba Hakone Tourist Information（静岡県御殿場市新橋1940-10）。
公开服务信息：营业 09:00–17:00；当天寄存 ¥600 / 件，过夜 ¥1,100 / 件，须在 17:00 前取回。
📌 这使路线顺畅：7/25 在御殿場駅寄大件 → 11:40 上山；7/26 下山回同一车站取回。建议出发前通过电话 +81-550-83-4770 或邮件 annai@gotemba.gr.jp 确认 7/25–7/26 周末大件行李的余位、尺寸与是否需预约。

🥾下午–傍晚：須走線登山（五合目 → 上江戸屋 / 本八合目）
12:40/14:00 → 傍晚 須走口五合目 → 上江戸屋（本八合目）· 須走线上坡
🗺️ 路线概要：
　• 起点：須走口五合目（海拔 ~2,000 m）
　• 终点：上江戸屋 / 本八合目 胸突江戸屋（Kami-Edo-ya / Munatsuki Edo-ya）（海拔约 3,370 m；公开资料口径略有差异）
　• 須走口五合目海拔约 2,000 m → 本八合目，海拔爬升约 1,350 m
　• 官方给出的須走线全程上山参考为 6.9 km / 约7小时到山顶；到上江戸屋不是山顶，建议按 4.5–6小时 保守预估（含休息、周末拥堵）

🏔️ 须走线特点：
　• 东侧路线，森林路段较长，视野开阔处可眺望骏河湾
　• 須走線在八合目附近与吉田線汇合，之后共用路径至山顶；上江戸屋位于汇合后的路段上（本八合目，公开资料多标为约 3,350–3,370 m），当晚住宿顺路
　• 次日仍原路沿須走線下撤回須走口五合目（注意八合目分岔口认准「須走口 →」，别误入吉田口下到北麓河口湖，详见 Day 3）
　• 相比吉田线人少，更安静

🏠 上江戸屋（本八合目・胸突江戸屋）：
　• 山小屋住宿（通铺/胶囊式床位）
　• 提供简单晚餐和早餐（需额外付费或含在房费中）
　• 有饮用水补给（瓶装水较贵，建议自备部分）
　• 务必提前预约！旺季非常紧张；请在预约页确认 7/25 当晚是否显示 ○（可订）
　• 🔗 预约：yamatan.net 上江戸屋预约页

⏰ 时间节奏参考（两种情况）：
　• 理想版：11:40 登山巴士 → 12:40 抵达五合目 → 13:00 开爬 → 18:00–19:00 抵达上江戸屋
　• 稳妥兜底版：13:00 登山巴士 → 14:00 抵达五合目 → 14:20 开爬 → 19:00–20:00 抵达上江戸屋（偏晚，需提前和山屋确认最晚入住/晚餐安排）
　• 上山当天务必在五合目完成 FUJI NAVI / 入山手续并保留头灯在随手位置
　• 到山屋后尽快吃饭、整理冲顶包，尽量睡到 01:00 前后
🚨 重要装备清单（必带）：
• 头灯（及备用电池）· 登山杖（必备！）· 保暖中层（抓绒/薄羽绒）· 冲锋衣裤（防风防水）· 手套· 帽子· 太阳镜· SPF50+防晒霜

• 登山鞋（防滑高帮最佳）

↘️清晨：須走線（Subashiri Trail）下撤 · 原路回須走口五合目
🔴 ⚠️ 下山最关键的一点：八合目下的岔路口，务必走「須走口方向」！

你要回的是上山起点 = 須走口五合目（御殿場登山巴士的那个五合目），所以下山走須走線下山道原路返回。

⚠️ 富士山最著名的下山迷路点就在这里：从山顶下来，須走線与吉田線共用同一条下山道直到八合目「下江戸屋」附近的分岔点。在此处必须认准指示牌走「須走口下山道 / Subashiri Trail →」（須走线为红色标识），切勿随大流走成「吉田口下山道」（会一路下到北麓河口湖，方向完全相反、回不了御殿场！）。

✅ 正确返程链路（简单直接）：
　1. 山顶 → 八合目分岔口 →（须走线下山道·含著名「砂走り」砂跑段）→ 須走口五合目 — 约 3–4h（官方参考约4h）
　2. 須走口五合目 →（须走线登山巴士）→ 御殿場駅 — 约 60min
　3. 御殿場駅 →（免费接驳）→ 御殿場奥莱取行李 — 约 10min
　4. 御殿場奥莱 → 成田机场（详见下方）

💡 相比绕北麓河口湖，这条原路返回省掉约 2 小时中转，且下山巴士班次多（末班 20:00），从容很多。
05:30 – 09:30/10:00 山顶 → 須走口五合目（須走線下山道）
🗺️ 从山顶沿下山道下撤，八合目分岔口认准「須走口 →」（见上方红框提醒）
⬇️ 官方路线参考：須走线下山道约 6.2 km / 约4小时；体能好且砂走り顺利可更快
🏃 中段有須走線特色的 「砂走り」（砂跑）——从七合目后开始的大段火山砂坡，建议穿高帮鞋+绑腿/护目镜/口罩防沙尘
